# Múltiplos de un Vector
***
Para multiplicar un Vector por un escalar basta con multiplicar cada término por el escalar ([[Multiplicar por escalar]]). 

>**Propiedades de los múltiplos**
>Todos los múltiplos de un vector poseen igual dirección, ángulo de inclinación, son paralelos y colineales. 
>$$\text{Dirección = Ángulo de Inclinación = Paralelos = Colineales = Múltiplos}$$

>**Magnitudes de los múltiplos**
>Con las magnitudes de estos múltiplos es lo mismo que tener la magnitud del vector original multiplicada por el valor absoluto del escalar.
>$a = \text{es un vector}$ $$||ka||=|k| \space||a||$$

>**Otras Reglas**
>1. Si se tiene un vector $a$, $ka$ y $k>0$ se conserva la dirección y sentido.
>2. Si se tiene un vector $a$, $ka$ y $k<0$ se conserva la dirección, pero con sentido opuesto.

>**Terminología**
>1. Si se multiplica al vector por un $-1<k<1$ se dice que el vector se *contrae/dilata*.
>2. Si se multiplica al vector por un $|k|>0$ se dice que el vector se *dilata*.
