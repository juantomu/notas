# Definición Producto Mixto
***
Existen restricciones para el [[Definición de Producto Vectorial|producto vectorial]], puesto que solo se puede dar en vectores de $R{^3}$, por ende, hay que tener cuidado al tratar de resolver algunas combinaciones del producto [[Definición de Producto Escalar|escalar]] y [[Definición de Producto Vectorial|vectorial]].

El producto mixto se define así:

$$a \cdot(b \times c)$$

> Otra forma común de escribirlo es:
>$$(a,b,c)$$

El **producto mixto** se puede resolver armando una matriz de 3x3 donde cada fila corresponde a las coordenadas de cada vector en el mismo orden, y se halla el determinante de forma sencilla utilizando [[Determinante#Matrices 3x3 - Regla de Sarrus|regla de Sarrus]].
